Klient do aplikacja do zarządzania licytacją brydżową z możliwością edycji odzywek. 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
